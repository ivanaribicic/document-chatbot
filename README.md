Document chatbot is a user-friendly research tool designed for effortless information retrieval. Users can input PDF files or URLs and ask questions to get relevant information.


<img width="1920" height="887" alt="app_chat" src="https://github.com/user-attachments/assets/161383d9-e15a-406d-b26a-0ed9536dfb2e" />


Features:
    
  Load URLs or upload PDF files
  
  Process document content, create text chunks and a vector store using FAISS, set a conversation chain
  
  Interact with the LLM (Hugging Face Mistral-7B-Instruct) by inputting queries and receiving answers along with source URLs
